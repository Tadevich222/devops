# kBot - Telegram Bot для курсу DevOps

Простий Telegram бот, створений на Go з використанням фреймворку Telebot для навчальних цілей курсу DevOps.

##  Посилання на бота

[https://t.me/courseSV_bot](https://t.me/courseSV_bot)

##  Опис

kBot - це навчальний Telegram бот, який демонструє базову взаємодію з користувачами через Telegram API. Бот приймає текстові повідомлення та відповідає на команди.

## Використання

### Доступні команди

- `/start` - Запуск бота та отримання привітального повідомлення
- Будь-який текст - Бот відповість та підтвердить отримання вашого повідомлення

### Приклади використання

```
Користувач: /start
Бот: Привіт! Я бот для курсу DevOps.

Користувач: Привіт, як справи?
Бот: Я отримав: Привіт, як справи?
```

## Технології

- [Go](https://golang.org/) - мова програмування
- [Telebot v4](https://github.com/tucnak/telebot) - фреймворк для Telegram Bot API
- [Cobra](https://github.com/spf13/cobra) - CLI фреймворк
- [godotenv](https://github.com/joho/godotenv) - завантаження змінних середовища
